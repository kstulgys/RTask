# Currency Exchange App

visit live at https://exchangeme.netlify.com/

## Getting Started

git clone ...repo

### Installing

Install netlify-cli

```
npm install netlify-cli -g
```

Intall dependencies

```
yarn or npm i
```

start project

```
yarn dev or npm dev
```

## Running the tests

```
yarn test
```

```
yarn cypress
```
