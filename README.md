# Currency Exchange App

Currency exchange sample app

## Getting Started

git clone ...repo

### Installing

You net to have netlify-cli installed globally

```
npm install netlify-cli -g
```

Intall dependencies

```
yarn or npm i
```

start project

```
yarn dev or npm dev
```

## Running the tests

```
yarn test
```

```
yarn cypress
```
